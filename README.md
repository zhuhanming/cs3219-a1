## Set-up

First, clone this repository:

```sh
git clone https://github.com/zhuhanming/cs3219-a1.git
```

Then, simply navigate into the directory and run Docker Compose:

```sh
cd cs3219-a1
docker-compose up
```

Head over to `http://localhost:3000` to see the website!
